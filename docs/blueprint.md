# **App Name**: ContentForge AI

## Core Features:

- Research-Driven Blog Generation: Multi-step tool that uses generative AI to analyze keywords, extract web insights, and produce deep-research, SEO-optimized articles with human-like prose.
- Cinematic AI Generation Experience: Interactive workflow with live-thinking animations, progress glows, and typing effects that visualize the AI synthesis process in real-time.
- SEO Intelligence Engine: A scoring tool that provides readability metrics, keyword density analysis, and intelligent SERP-focused suggestions powered by LLM logic.
- Pro-Editor Workspace: Next-gen rich text editor with markdown support, glassmorphic floating toolbars, and seamless split-view for AI content refinement.
- Firebase Document Cloud: Real-time synchronization for blogs, drafts, and analytics reports utilizing Firestore for lightning-fast workspace state persistence.
- Context-Aware Optimization Toolkit: One-click tool for rewriting sections, shifting tone, generating social media assets, and creating AI thumbnail prompts based on article content.
- Performance Insight Dashboard: Advanced UI with SEO score charts, AI usage metrics, and history logs designed with high-end dark-mode aesthetics and smooth transitions.

## Style Guidelines:

- Primary color: Vibrant electric violet (#9163FF) for high-tech brand identity. Background: Deep matte charcoal (#0D0B12) with a slight blue-violet undertone for a cinematic depth.
- Accent color: Bright cyan (#3B71F7) used for high-contrast highlights, glow effects, and interactive call-to-action components.
- Font pairing: 'Space Grotesk' for tech-forward headlines and 'Inter' for highly legible body text and UI controls.
- Advanced layout utilizing animated grids, glassmorphism card components with varying backdrop blur levels, and floating sidebar navigation for a workspace feel.
- Thin-line neon icons with subtle gradient strokes that complement the futuristic SaaS theme.
- Extensive use of Framer Motion for micro-interactions: magnetic button physics, glowing loading bars, and shimmering transition effects.
